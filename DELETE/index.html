Exercice DELETE

DELETE from permet de supprimer des lignes.

DELETE from nom_de_la_table

WHERE nom_de_colonne OPERATEUR "valeur"

[and|or "nom_de_colonne" OPERATEUR "valeur"]

;

[ ] = optionnel

Exemple sur PHPmyadmin -> la base de données mydata

Selectionner la table de données animaux

Supprimer les deux lignes :

DETELE from animaux WHERE categorie='chat';
