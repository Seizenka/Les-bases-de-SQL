Drill 1

Utilise la zone éditable à droite pour écrire tes premières requêtes SQL.

Trouve la syntaxe pour effectuer chacune des requêtes suivantes (lis bien la formulation, chaque mot a son importance!). Vérifie

Retourne un tableau contenant

  • uniquement les villes : SELECT ville FROM Météo;
  • toutes les villes et leur température maximum : SELECT ville, haut FROM Météo;
  • toutes les villes et leur température minimum : SELECT ville, bas FROM Météo;
  • Là où les villes dont la température maximale dépasse 27 degrés : SELECT ville, haut FROM Météo WHERE haut > 27;
  • Là où les villes dont la température minimale sera inférieure ou égale à 15 degrés : SELECT ville, bas FROM Météo WHERE bas < 15;

  • Là où les villes dont la température minimale ne sera PAS égale à 15 degrés : SELECT ville, bas FROM Météo WHERE bas <> 15;
  • Là où les villes dont le nom commence par "Br" : SELECT ville FROM Météo WHERE ville LIKE 'Br%' ;
  • Là où les villes dont la température maximale se situe entre 26 et 28 degrés : SELECT ville, haut FROM Météo WHERE haut >= 26 ;
  • Là où les villes dont la température minimale est égale à 14 ou est égale à 16 degrés : SELECT ville, bas FROM Météo WHERE bas <= 16 ;
  • Là où les villes dont la température maximale est supérieure ou égale à 26 et la température minimale est inférieure à 14 : SELECT ville, bas, haut FROM Météo WHERE bas < 14 AND haut >= 26;

    Trouve la syntaxe pour effectuer chacune des requêtes suivantes:

    • Affiche tous les octocats et leurs données : SELECT * FROM octocats

    • Affiche le prénom, le nom et l'année de naissance des octocats de la promo "promo1-anderlecht" : SELECT lastname, firstname, birthdate FROM octocats WHERE promo='promo1-anderlecht';

    TRIER : ORDER BY

    Tu peux demander à MySQL de trier les résultats selon un ordre que tu définis. Pour cela, ajoute à la fin de ta requête: ORDER BY suivi de la colonne à utiliser pour le tri, suivi soit de ASC ("ascending order", de A à Z par exemple ou de la moins grande valeur à la plus grande valeur) ou de DESC ("descending order", de Z à A).

    Ex:

    SELECT ville FROM météo ORDER BY ville ASC;

    LIMITER LE NOMBRE DE RÉSULTATS

    Tu peux dire à SQL de limiter le nombre de résultats, en ajoutant le mot-clef LIMITsuivi de la rangée de départ (on commence à 0) puis une virgule, puis le nombre de rangées à garder.

    Par exemple, pour ne garder que la rangée du dessus, on ajoute LIMIT 0,1 à la requête:

    SELECT ville FROM météo ORDER BY ville ASC LIMIT 0,1 ;

    Drill 3

    • Affiche le prénom et nom de tous les octocats, par ordre alphabétique ascendant du prénom : SELECT firstname, lastname FROM octocats ORDER BY firstname ASC;
    • Affiche le prénom et nom de tous les octocats, par ordre alphabétique descendant du nom de famille : SELECT firstname, lastname FROM octocats ORDER BY lastname DESC;
    • Affiche le prénom, nom et âge de tous les octocats, du plus jeune au plus âgé : SELECT firstname, lastname, birthdate FROM octocats ORDER BY birthdate ASC;
    • Affiche le prénom, nom et âge de tous les octocats, du plus âgé au plus jeune : SELECT firstname, lastname, birthdate FROM octocats ORDER BY birthdate ASC;
    • Affiche le prénom, nom et âge de tous les octocats, du plus jeune au plus âgé, de la promo "promo1-central" : SELECT firstname, lastname, birthdate FROM octocats WHERE promo='promo1-central' ORDER BY birthdate ASC;
    • Affiche le prénom, nom et âge de tous les octocats, du plus jeune au plus âgé, de la promo "promo1-central" et dont l'âge se situe entre 23 et 28 ans : SELECT firstname, lastname, age FROM octocats WHERE promo='promo1-central' AND age BETWEEN 23 AND 28 ORDER BY age ASC;
    • Affiche le prénom, nom, âge et genre de tous les octocats, du plus jeune au plus âgé, de la promo "promo1-central" et dont l'âge se situe entre 25 et 35 ans : SELECT firstname, lastname, age FROM octocats WHERE promo='promo1-central' AND age BETWEEN 23 AND 35 ORDER BY age ASC;
    • Affiche le prénom, nom, âge et genre de tous les octocats, du plus jeune au plus âgé, de la promo "promo1-central", dont l'âge se situe entre 25 et 35 ans et dont le genre est masculin : SELECT firstname, lastname, age FROM octocats WHERE promo='promo1-central' AND age BETWEEN 23 AND 35 AND gender='M' ORDER BY age ASC;
    • Affiche le prénom, nom, âge de l'octocat le plus âgé de Becode/Central. Donc tu dois obtenir un tableau avec une seule rangée : SELECT firstname, lastname, age FROM octocats WHERE promo='promo1-central' AND age >=49 AND gender='M';
